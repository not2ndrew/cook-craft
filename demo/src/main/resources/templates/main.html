About

CookCraft is my second project created using frontend languages HTML, CSS, Javascript alongside with backend languages such as Java and MYSQL with a Java framework Spring Security. This project is a new years resolution for myself. This project was started at January 1, 2025 -> present.

CookCraft is a forum that allows registered users to create recipes for others to view. Users can also rate other user's recipes on a scale of 5-stars. The recipe will be rated on the average. The reviewer can also write down a comment on the user's recipe.
